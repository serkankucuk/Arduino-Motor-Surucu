# Arduino-Motor-Surucu
Arduino'da motor sürücü kullanarak hava sıcaklığına göre pervaneleri kontrol edebildiğimiz bir çalışma.

#### Kullanılan Bileşenler:
1. Arduino Uno
2. L298N motor sürücü
3. Led
4. Buzzer
5. 9-12v fan
6. Breadboard
7. Direnç
8. Çeşitli bağlantı kabloları
